# Sync State Engine

A lightweight API caching and state synchronization layer for React.

##  Features
- Smart caching
- Avoid duplicate API calls
- Optimistic updates
- Simple API

##  Why not React Query?
This project demonstrates how caching & sync works internally.

## Example
\`\`\`js
const { data } = useQuery("todos", fetchTodos);
\`\`\`

##  Concepts Covered
- State management
- Cache invalidation
- Subscription model
